# BEEditor

A lightweight terminal-based code editor written in **Go**, built using the **tcell** library.  
The project focuses on learning low-level terminal rendering, editor architecture, and text buffer manipulation.

This editor is inspired by minimal editors such as **kilo** and **vim-like interfaces**, but implemented from scratch as a learning project.

---

## Features

- Terminal UI powered by **tcell**
- Text buffer with editable lines
- Insert mode editing
- Command line mode (`:` commands like in vim)
- File loading and saving
- Scrollable view
- Cursor movement using arrows
- Line numbers
